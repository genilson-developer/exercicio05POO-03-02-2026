# Sistema de Cardápio

Mini sistema desenvolvido para a disciplina de Programação Orientada a Objetos.

O sistema permite:

- Cadastrar refeições
- Pesquisar refeições
- Remover refeições
- Calcular preço médio
- Listar refeições

Tecnologias utilizadas:

- Java
- Collections (Map, List)
- Streams
- Serialização de objetos
- JUnit para testes

Os dados são armazenados em um HashMap e podem ser persistidos em arquivo utilizando serialização de objetos.
